# BDII_Final_2024

## Iniciar
Se puede iniciar el backend, la base de datos y el frontend con los siguientes comandos:

### Base de datos
- `cd PencaAPI`
- `./dbup.sh`

### Backend
- `cd PencaAPI`
- `./apiup.sh`

### Frontend
- `cd PencaFront`
- `docker compose -f ./docker-compose.yml up --build`
